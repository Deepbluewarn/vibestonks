# 주간 하드 리셋 (누적 잔고 없음)

매주 월요일 09:00에 모든 트레이더의 잔고가 1,000으로, 모든 종목의 발행 주식 수가 0으로, 가격이 100으로 일괄 리셋된다. 누적 잔고 개념 없음. 한 주 = 자기완결적 미니 토너먼트.

## Considered Options

- **누적 잔고 + 주간 풀 균등 분배 + 월간 하드 리셋**: 두 단계 리셋 구조. 누적 자랑은 가능하지만 격차 누적·디플레이션·"부자가 더 부자됨" 문제. 두 단계 룰 자체가 토이 컨셉(가벼움)과 충돌.
- **누적 잔고만**: 한 번 망한 사람이 영영 못 따라잡음 → 의욕 상실.
- **매일 8시 리셋**: 데이트레이딩만 가능. 한 사이클이 너무 짧아 매수→매도 사이클 만들기 어려움.

## Consequences

- 매주 새 출발이라 진입 장벽 없음 (지난 주 결과가 다음 주 영향 X).
- "누적 자랑" 메커니즘은 없음 — 매주 1등이 다른 사람이어도 그 기록은 별도 페이지(명예의 전당 등)에서만 의미를 가짐.
- 주간 풀 균등 분배 같은 보조 메커니즘 불필요 (어차피 다 0으로 가니까).
