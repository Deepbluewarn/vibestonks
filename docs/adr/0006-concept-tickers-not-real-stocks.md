# 컨셉 라벨 종목 (실제 주식명 미사용)

종목 이름은 일상 사건 모티프의 농담 라벨(`$오늘비왔다`, `$팀장화남`, `$rain` 등)이며, 실제 상장사 이름은 의도적으로 사용하지 않는다.

## Considered Options

- **KOSPI 200 등 실제 상장사 이름**: 처음엔 가장 익숙한 농담이지만 (a) 상표권 침해 시비, (b) 자본시장법상 진짜 시세로 오인될 위험, (c) 해당 기업 평판에 의도치 않은 피해 가능. 초기 시드(KOSPI 127개) 코드는 git history에 남아 있음.
- **추상 라벨 (`$알파`, `$브라보`)**: 법적으로 가장 안전하나 종목 정체성이 사라져 농담 자체가 없어짐.
- **사용자 제안형**: 가장 활기 있지만 큐레이션·신고 시스템 등 운영 부담이 큼.

## Consequences

- `lib/seed-data/concepts.ts`의 풀을 자유롭게 늘리거나 교체 가능 — 단일 파일에 격리됨.
- 풀에 새 라벨을 추가하려면 PR 하나만 보내면 됨.
- 컨셉 라벨은 농담의 보편성이 높아 다양한 그룹에서 작동.
