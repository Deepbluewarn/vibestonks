# AMM with linear bonding curve (호가창/오더북 없음)

가격 발견 메커니즘으로 호가창(order book) 대신 AMM을 채택. 본딩 커브는 가장 단순한 선형 형태 `price = 100 + 2 × outstanding_shares`. 매수 시 비용 = 본딩 커브의 적분, 매도 시 수익 = 같은 적분. 항상 거래 가능하고 가격이 결정론적으로 도출됨.

## Considered Options

- **호가창**: 진짜 주식의 본 모습이지만 매칭 엔진·양방향 유동성 필요. 토이엔 압도적 과잉.
- **Constant product (Uniswap x·y=k)**: 두 자산 풀이 필요해 "포인트 vs 종목" 페어로 모델링 가능하지만, 선형보다 직관 ↓ 구현 ↓. 가격 변동의 직관성이 핵심인 토이엔 선형이 더 적합.
- **고정 가격**: 가격이 안 움직이면 그건 주식 시뮬레이션이 아니라 단순 도박(예측 시장)이 됨.

## Consequences

- 공매도 없음. 지정가 없음. 시장가 매수/매도만 존재. 룰 설명 1줄.
- "발행 주식 수"는 가격에서 즉시 역산되므로 비공개 의미 없음 — 둘은 사실상 같은 정보.
- 한 사람이 단독으로 매수→매도 왕복하면 손익 0 (다른 거래가 끼지 않을 때). 진짜 손익은 **다른 트레이더가 그 사이에 곡선을 움직였느냐**에서만 발생.
