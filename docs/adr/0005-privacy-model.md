# 프라이버시 모델 (포지션 비공개, 상위 3명만 랭킹 공개)

다른 트레이더의 보유 포지션·잔고·체결 내역은 모두 비공개. 공개되는 정보는 (a) 모든 종목의 현재 가격과 가격 차트 (체결자 익명), (b) 라운드 마감 직후 랭킹 상위 3명의 이름+최종 잔고, (c) 본인 등수(상위 3명 외)뿐.

## Considered Options

- **전체 공개 리더보드**: 자극적이지만, 친구 단위의 작은 토이라 잔고로 사람을 평가하는 분위기가 생길 우려. 작은 그룹일수록 위험.
- **포지션도 공개**: 따라하기 게임으로 변질 (셸링 포인트 효과 ↑). 자기 의지로 결정하는 재미가 사라짐.
- **풀 익명**: 누가 1등인지도 모르면 게임의 사회적 보상이 사라짐 — 농담거리가 없음.

## Consequences

- 포지션 비공개는 잔고 비공개로 자동 확장됨: 현재가가 공개되므로 `잔고 = 시작 잔고 + 거래 손익`이 유추 가능. 따라서 잔고도 본인 외 비공개.
- 랭킹 발표 시점에서만 상위 3명의 잔고가 노출됨 (그 한 순간의 스냅샷).
- 체결 내역 비공개 → 가격 차트는 누가 했는지 모르는 단순 시계열로 표시.
